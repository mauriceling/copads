reverseBlock(self, block, rtype='string')

source code 

Function to reverse a string or a list.

>>> import jigsaw
>>> c = jigsaw.JigsawCore()
>>> c.reverseBlock('1234567890', 'string')

>>> c.reverseBlock('1234567890', 'list')

>>>

  • rtype (string) - return type. Allowable values are 'string' (return type will be a string) or 'list' (return type will be a list). Default = 'string'.

reversed block

shuffleBlock(self, block, length, rtype='string')

source code 

Function to take a block, divide the block into smaller sub-blocks, and shuffle the order of the sub-blocks.

>>> import jigsaw
>>> c = jigsaw.JigsawCore()
>>> text = 'MyNameIsMauriceLing,AndIInventedJigsawEncryption'
>>> (shuffled_blocks, block_order) = c.shuffleBlock(text, 10, 'string')
>>> for i in range(len(shuffled_blocks)):
...     print block_order[i], shuffled_blocks[i]
... 
3 edJigsawEn
0 MyNameIsMa
4 cryption
1 uriceLing,
2 AndIInvent
>>>
Parameters:
  • block (string or list) - data block to be shuffled.
  • length (integer) - length of sub-block.
  • rtype (string) - return type. Allowable values are 'string' (return type will be a string) or 'list' (return type will be a list). Default = 'string'.
Returns:
(shuffled block, order of sub-blocks)
