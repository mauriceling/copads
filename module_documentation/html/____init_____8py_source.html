1 """!
2 COPADS (Collection of Python Algorithms and Data Structures).
3 
4 The main aim of COPADS is to develop a compilation of of Python data
5 structures and its algorithms, making it almost a purely developmental
6 project. Personally, I look at this as a re-usable collection of tools
7 that I can use in other projects. Therefore, this project is
8 essentially "needs-driven", except a core subset of data structures
9 and algorithms.
10 
11 This project originated from 3 threads of thought. Firstly, while
12 browsing through Mehta and Sahni's Handbook of Data Structures and
13 Applications, I thought there might be utility to have a number of the
14 listed data structures implemented in Python. Given my interest in
15 biological data management, having a good set of data structures is
16 always handy. The 2nd thread of thought came from Numerical Recipes.
17 Again, I thought these algorithms will be handy to have and had
18 started to translate some of them into Python during some overly
19 energetic days. Finally, Python Cookbook had undergone 2 editions by
20 2008 and ActiveState had provided an online platform for Python
21 Recipes which I found to be useful and can see how some of these
22 recipes can be merged. Thus, COPADS is borned.
23 
24 Project website: https://github.com/copads/copads
25

27 those adapted, are covered under Python Software Foundation License
28 version 2.
29 """
